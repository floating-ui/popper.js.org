# React Native

**Status: Experimental**

Popper 3's platform-agnostic architectural rewrite means it now supports React
Native. The support for React Native is brand new, so it's currently on `v0` to
refine the API based on people's usage of the library.

## Install

```shell
# With npm
npm i @popperjs/react-native

# With Yarn
yarn add @popperjs/react-native
```

## Usage

Pass the `reference` and `popper` variables to the `ref` prop of each element
respectively. `x` and `y` represent the positioning coords.

```jsx
import { View, Text } from 'react-native';
import { usePopper } from '@popperjs/react-native';

function App() {
  const { x, y, reference, popper } = usePopper();

  return (
    <View>
      <View ref={reference}>
        <Text>Reference</Text>
      </View>
      <View
        ref={popper}
        style={{
          position: 'absolute',
          left: `${x}px`,
          top: `${y}px`,
        }}
      >
        <Text>Popper</Text>
      </View>
    </View>
  );
}
```

### Return value

`usePopper` returns the following object:

```flow
type UsePopperReturn = {|
  // @popperjs/core `computePosition()` return type, includes x & y coords,
  // `modifiersData`, and `strategy`.
  ...ComputePositionReturn,
  placement: ?Placement,
  // The popper's `offsetParent` if not contained within the base App root.
  offsetParent: {| current: any |},
  popper: {| current: any |},
  reference: {| current: any |},
  // Apply to the <ScrollView /> in which the reference element is contained,
  // if the popper is not also within it.
  scrollProps: {|
    onScroll: (event: {
      nativeEvent: {
        contentOffset: {| x: number, y: number |},
      },
    }) => void,
    scrollEventThrottle: 16,
  |},
|};
```

### Options

These options can be passed to `usePopper`:

```flow
type Options = {
  placement: Placement, // default: "bottom"
  modifiers: Array<Modifiers>, // default: []
  // If the Popper is contained within the same <ScrollView /> as the
  // reference element.
  sameScrollView: boolean, // default: true
};
```

Example usage if the popper is within the `<App />` root, while the reference
element is inside a `<ScrollView />`:

```jsx
import { View, Text, ScrollView } from 'react-native';
import { usePopper } from '@popperjs/react-native';

function App() {
  const { x, y, reference, popper, scrollProps } = usePopper({
    sameScrollView: false,
  });

  return (
    <View>
      <ScrollView {...scrollProps}>
        <View ref={reference}>
          <Text>Reference</Text>
        </View>
      </ScrollView>
      <View
        ref={popper}
        style={{
          position: 'absolute',
          left: `${x}px`,
          top: `${y}px`,
        }}
      >
        <Text>Popper</Text>
      </View>
    </View>
  );
}
```
