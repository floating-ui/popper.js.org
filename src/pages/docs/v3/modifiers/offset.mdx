# offset

The `offset` modifier displaces the popper element from the reference element.

<Offset />

## Usage

```js
import { computePosition, offset } from '@popperjs/dom';

computePosition(reference, popper, {
  modifiers: [offset()],
});
```

## Options

`offset()` can take either:

- a number, representing the `mainAxis` (distance) in pixels of displacement
- an object, containing `mainAxis` and `crossAxis` properties to configure both
  axes
- a function, taking the `reference` and `popper` rects and the current
  placement, and return either of the above

```js
// number
offset(10);

// object
offset({ mainAxis: 25, crossAxis: 5 });

// function
offset(({ popper, reference }) => -popper.height);
```

## Order

In most cases, this should be the first modifier in your `modifiers` array.
