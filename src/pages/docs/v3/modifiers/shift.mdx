# shift

The `shift` modifier shifts the popper element to prevent it from overflowing
clipping boundaries.

<Shift />

Scroll the container.

## Usage

```js
import { position, shift } from '@popperjs/dom';

position(reference, popper, {
  modifiers: [shift()],
});
```

## Options

`shift()` takes an object of options:

```flow
type Options = {
  mainAxis: boolean, // default: true
  crossAxis: boolean, // default: false
  ...DetectOverflowOptions,
};
```

### `mainAxis`

Describes whether to shift the popper along the main axis. For vertical
placements, this is the `x` axis. For horizontal ones, it's the `y` axis.

```js
shift({
  mainAxis: false, // true by default
});
```

### `altAxis`

Describes whether to shift the popper along the alt axis. For vertical
placements, this is the `y` axis. For horizontal ones, it's the `x` axis.

Enabling this can lead to the popper overlapping the reference element, and is
generally replaced by the `flip` modifier.

```js
shift({
  crossAxis: true, // false by default
});
```

### `...detectOverflowOptions`

All of [`detectOverflow`](../../utils/detectOverflow/#options)'s options can be
passed.
